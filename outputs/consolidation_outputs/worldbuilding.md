# worldbuilding：花海咲季相关世界观设定

> 范围：本文件仅整理输入材料中与花海咲季相关、可作为 RAG 检索条目的稳定世界观设定与剧情事实。证据以 `scene_id` 标注。为避免噪音，不收录明显玩笑式、特殊演出式或与主线世界观无关的临时设定；不复制长段原始台词。

## 0. 术语与证据约定

- **初星学园**：培养偶像与制作人的学园，设有偶像科、制作人科，并存在高等部、中等部、普通科等相关学生生活结构。咲季是初星学园偶像科学生，并以入学考试第一名身份入学。证据：`scene-00066`、`scene-00078`、`scene-00172`、`scene-00200`、`scene-00242`
- **咲季 / 花海咲季**：以“世界第一偶像”“未来顶级偶像”为目标的偶像。她拥有优秀的运动、学业、容貌与自我管理能力，但同时有早熟型成长瓶颈、强烈好胜心与不愿示弱的问题。证据：`scene-00172`、`scene-00174`、`scene-00430`
- **佑芽 / 花海佑芽**：咲季的妹妹，也是咲季最重要的憧憬对象、对手与共同成长者。二人的关系既是姐妹，也是偶像竞技中的宿敌关系。证据：`scene-00173`、`scene-00203`、`scene-00222`、`scene-00498`
- **N.I.A / NEXT IDOL AUDITION**：偶像争夺粉丝与排名的选拔活动，最终日有大型公开试镜 `FINALE`。咲季与佑芽约定在最终日直接对决。证据：`scene-00183`、`scene-00184`、`scene-00211`
- **FINALE**：N.I.A 最大规模的最终公开试镜舞台，也是咲季、佑芽、燐羽等人的重要对决节点。证据：`scene-00183`、`scene-00184`、`scene-00186`
- **H.I.F / Hatsuboshi IDOL FESTIVAL / 初星偶像节**：输入材料中作为决定学园顶点与“一等星”的大型偶像赛事或庆典体系出现，包含组合部门、个人部门、冠军与“一等星”称号等要素。证据：`scene-00205`、`scene-00270`、`scene-00448`、`scene-00494`、`scene-00497`、`scene-00500`
- **一等星**：学园顶级偶像称号，常与 H.I.F 或初星偶像祭结果关联。咲季曾以此为最高目标，并在后续被宣布为夏季“一等星”。证据：`scene-00173`、`scene-00193`、`scene-00500`
- **Campus mode!!**：初星学园偶像的重要传统曲目，也被称为学园偶像的“必杀技”。证据：`scene-00024`、`scene-00040`
- **Re;IRIS**：咲季、手毬、琴音相关组合名，由 `Re;` 与 `IRIS` 意象组成，强调坚韧、女神、彩虹、回馈希望与治愈。证据：`scene-00479`、`scene-00497`
- **Begrazia**：与 Re;IRIS 对决的新偶像组合，成员包括星南、佑芽、美铃，理念偏向全员压倒性实力。证据：`scene-00483`、`scene-00490`、`scene-00497`
- **SSD**：咲季特制的训练/恢复饮品或配方体系，常用于塑形、恢复精神或打造偶像体质，味道和外观常被吐槽。证据：`scene-00063`、`scene-00079`、`scene-00257`、`scene-00259`、`scene-00411`

## 1. 通用设定：初星学园

### 1.1 学园定位与组织

- 初星学园是偶像培育学校，学生以成为学园第一、顶级偶像、世界第一偶像等目标进行学习、训练、演出与选拔。证据：`scene-00024`、`scene-00172`、`scene-00242`
- 学园设有偶像科与制作人科，偶像与制作人之间存在正式的培育关系；制作人需制定方案、记录培育过程、引导偶像成长。证据：`scene-00200`、`scene-00175`、`scene-00290`
- 初星学园有中等部与高等部，也存在内部升学组与外部入学的新生竞争。咲季属于从高中阶段进入初星学园的高等部新生。证据：`scene-00062`、`scene-00066`、`scene-00078`、`scene-00166`、`scene-00242`
- 入学考试存在综合排名、笔试等评价维度；咲季是入学考试综合第一名，广曾提及自己笔试满分。证据：`scene-00078`、`scene-00172`、`scene-00242`
- 学园存在候补录取制度，佑芽以候补资格入学，需要通过面谈、名册登记与选拔考试证明实力。证据：`scene-00200`、`scene-00254`、`scene-00474`
- 学园学生会不仅处理校务，也深度参与偶像活动、预算、培育与后辈管理。星南、燕、莉波、佑芽等人与学生会有关。证据：`scene-00160`、`scene-00161`、`scene-00254`、`scene-00279`
- 普通科与偶像科的界限并非绝对，学园生活中前辈会跨科别照顾后辈。证据：`scene-00279`

### 1.2 校园设施与地理

- 初星学园宿舍有中庭，可用于讨论宿舍活动，但涉及噪音、许可与人员安排等问题。证据：`scene-00001`
- 学园校舍后方有森林与竹林，校内禁止随意砍伐树木。证据：`scene-00001`
- 学园内或附近有可玩水的河滩/河边，是学生休息日与夏日活动地点。证据：`scene-00119`、`scene-00120`
- 学园有操场，咲季与佑芽小时候曾在运动会赛跑的记忆与此关联。证据：`scene-00106`
- 初星学园拥有训练室，可被偶像借用进行练习；训练室也是咲季与佑芽比试、团队训练、特训的常用场所。证据：`scene-00173`、`scene-00202`
- 初星学园训练设施齐全，咲季入学理由之一就是校内设备完备，甚至接近她家中的训练环境。证据：`scene-00104`、`scene-00114`
- 学园食堂提供“偶像盖饭”等健康营养取向的菜品；咲季会向参观者推荐。证据：`scene-00146`
- 学园有开放式 Live 舞台，可供偶像自由演唱、即兴表演或举办小型舞台活动。证据：`scene-00278`
- 学园有学生会室，学生会室会进行大扫除、桌椅更换、资料整理等日常管理。证据：`scene-00160`、`scene-00161`
- 宿舍或学园生活中存在食堂、厨房、学生房间、训练器具、健身设备等日常空间，偶像生活与训练高度交织。证据：`scene-00460`、`scene-00114`、`scene-00154`

### 1.3 宿舍与共同生活

- 初星寮是学园宿舍体系的重要空间，宿舍长负责照顾成员生活、记录日常、组织活动。证据：`scene-00279`、`scene-00001`
- 初星寮有传统章鱼烧烤盘，维护铁板是寮长工作的一部分，宿舍会举办章鱼烧派对等集体活动。证据：`scene-00159`
- 宿舍成员之间会互相照顾生病者、分担工作，也会准备慰问品或特制饮品。证据：`scene-00287`、`scene-00288`
- 组合成员可能被要求共同生活以增进理解，并在共同生活中决定 C 位、队长、组合名与训练分工。证据：`scene-00459`、`scene-00460`
- 咲季在组合共同生活中坚持亲自做饭，理由是宿舍食堂无法满足每个成员的专属膳食需求。证据：`scene-00460`
- 宿舍生活会改变成员关系：争执、饮食矛盾、学费压力、深夜谈心、晨练等事件会推动组合成员从竞争转向互助。证据：`scene-00462`、`scene-00463`、`scene-00470`

### 1.4 课程、学业与训练体系

- 偶像学生需兼顾学业与偶像活动；成绩不佳会限制偶像活动。咲季会督促佑芽学习。证据：`scene-00206`、`scene-00124`
- 学园允许学生提前学习未接触课程，例如咲季请求学习尚未接触的数学 B 内容。证据：`scene-00332`
- 学园有声乐课，包含发声练习与课题曲练习；咲季与手毬会争抢练习机会。证据：`scene-00062`
- 学园有家政课，学生可制作点心并跨班级交流。证据：`scene-00147`
- 学园训练体系包含舞蹈、声乐、形象/表现力、体能、饮食、心理调整等多方面。证据：`scene-00072`、`scene-00083`、`scene-00084`、`scene-00340`
- 舞蹈训练、声乐训练、表现力训练是相对独立的训练项目；训练室被占用时，可转往卡拉 OK 进行声乐训练。证据：`scene-00072`
- 表现力训练可包含化妆、服装、应援等多方面内容，用于提升偶像可爱度与粉丝反应。证据：`scene-00084`
- 瑜伽、坐禅、拉伸、核心肌群训练等被用于精神稳定、专注力提升、身体冷却与表演准备。证据：`scene-00083`、`scene-00148`、`scene-00156`、`scene-00341`
- 训练后拉伸用于缓解乳酸堆积，训练后冷静身体被视为重要环节。证据：`scene-00148`
- 过度训练会带来嗓子受伤、肌肉酸痛、发烧、旧伤复发等风险，训练与休息需要平衡。证据：`scene-00409`、`scene-00333`、`scene-00265`、`scene-00267`
- 初星训练重视“科学训练”和“传统修行”的结合；咲季与燕曾讨论将最新器材训练与古老修行取长补短。证据：`scene-00158`

## 2. 通用设定：演出、选拔、粉丝与偶像活动

### 2.1 演出与歌曲

- 初星学园定期公演名为『初』，出演资格需通过中期与终期两次考核，终期结果决定能否获得主场演唱会举办权。证据：`scene-00305`
- 《Campus mode!!》是初星学园偶像在重要时刻演唱的传统曲目，也被称为初星学园偶像的“必杀技”。证据：`scene-00024`、`scene-00040`
- 初星学园偶像服装可被称为“战斗服”，舞台服装与传统曲目共同承载学园身份。证据：`scene-00024`、`scene-00025`
- 偶像拥有原创歌曲被视为成熟标志之一。咲季曾主动向制作人索要兼具可爱、帅气与成熟性感的原创曲。证据：`scene-00012`
- 歌曲可以是量身打造，也可以由偶像通过实力与诠释“夺回”或“据为己有”。咲季曾要求把不完全属于自己的歌改得更适合自己。证据：`scene-00034`、`scene-00038`
- 偶像共演不只是同台，而是让粉丝获得不同体验、实现粉丝群体共享，并促使偶像互相吸收优点。证据：`scene-00007`、`scene-00008`、`scene-00009`
- 排练也可被视为献给粉丝与伙伴的演出；咲季在排练中会以正式演出标准投入。证据：`scene-00035`
- 演唱会、迷你演唱会、开放舞台、谈话环节、握手会、粉丝见面会、签名会、直播、SNS 视频等都属于偶像活动范围。证据：`scene-00025`、`scene-00256`、`scene-00258`、`scene-00383`、`scene-00224`

### 2.2 选拔、考试与赛事

- 初星学园存在校内偶像选拔赛与组合选拔考试，偶像需通过选拔争取出演、参赛、加入组合或获得更大舞台机会。证据：`scene-00048`、`scene-00290`、`scene-00305`、`scene-00474`
- 选拔考试可作为加入组合或证明实力的门槛；佑芽被要求参加选拔考试并取得成绩后，才可正式加入相关竞争。证据：`scene-00474`
- 选拔结果对后续演唱会表现直接相关，偶像可在合格后继续修正失误、准备演唱会。证据：`scene-00401`
- N.I.A 是面向偶像的粉丝争夺与排名活动，设有粉丝投票排行榜，并在最终日举行最大规模公开试镜 FINALE。证据：`scene-00183`、`scene-00184`、`scene-00211`
- FINALE 是咲季与佑芽约定最终对决的舞台，也是燐羽等强敌参与对决的节点。证据：`scene-00183`、`scene-00186`
- 极月学园是初星学园的竞争对手学校，由多家事务所相关势力支持，其精锐偶像会在 N.I.A 等赛事与初星学园多次交锋。证据：`scene-00181`、`scene-00185`、`scene-00209`、`scene-00212`
- H.I.F 是决定学园顶点和“一等星”的重要偶像赛事/庆典体系。输入中既有夏季大赛、冬季 H.I.F，也有 Hatsuboshi IDOL FESTIVAL 与初星偶像节的表述，应按“初星大型偶像竞技庆典体系”检索。证据：`scene-00205`、`scene-00219`、`scene-00270`、`scene-00494`、`scene-00500`
- H.I.F/初星偶像祭可包含组合部门与个人部门；组合部门通过投票或综合评价选出冠军，个人赛冠军与“一等星”称号相关。证据：`scene-00448`、`scene-00497`、`scene-00499`、`scene-00500`
- Re;IRIS 在初星学园偶像祭决赛中击败 Begrazia，成为 H.I.F 冠军组合；随后咲季被宣布为夏季“一等星”。证据：`scene-00497`、`scene-00499`、`scene-00500`

### 2.3 粉丝、排名与人气机制

- 偶像行业存在知名度、粉丝数量、投票排名与收入/机会挂钩的机制。证据：`scene-00168`、`scene-00183`、`scene-00184`
- 粉丝支持可被视为偶像力量来源；制作人曾把粉丝比作外接肌肉组织。证据：`scene-00185`、`scene-00192`、`scene-00199`
- 粉丝不仅是观众，也会成为偶像力量的一部分；偶像需向粉丝传达心意并从反馈中获得力量。证据：`scene-00042`、`scene-00199`
- 咲季的粉丝群体眼光挑剔，尤其重视舞蹈实力。证据：`scene-00007`
- 偶像可通过 SNS、直播、上传视频、电视采访、粉丝见面会等手段提升曝光，但也存在炎上、误传与现场失控风险。证据：`scene-00256`、`scene-00257`、`scene-00310`
- 握手会会聚集大量粉丝，是偶像展示魅力与留下记忆的重要互动活动。证据：`scene-00383`、`scene-00384`

### 2.4 校外工作与地方活动

- 偶像工作不限于舞台演出，也包括一日店长、商店街活动、动漫店接待、观光宣传、综艺节目、商品推销、导览语音、温泉店体验、露营场宣传等。证据：`scene-00022`、`scene-00255`、`scene-00352`、`scene-00358`、`scene-00371`、`scene-00375`
- 咲季曾担任天川市观光大使，职责是通过宣传视频与景点推荐吸引游客；初星学园也被作为可推荐的旅游点。证据：`scene-00359`、`scene-00360`
- 咲季曾在孩子相关活动中任体操课老师、应援练习带领者、徒步拉力赛巡视者，体现偶像面向儿童与地方活动的社会角色。证据：`scene-00349`、`scene-00350`、`scene-00363`、`scene-00364`、`scene-00365`、`scene-00366`
- 地方演出如名古屋公演会涉及临时领队、歌单调整、人员变动与日程协调；咲季曾临危受命担任代理领队。证据：`scene-00281`
- 偶像工作也涉及打工与兼职现实：琴音因学费压力打工，清夏曾在夏日演唱会会场售卖商品，学园圣诞活动也需要兼职与幕后人手。证据：`scene-00245`、`scene-00247`、`scene-00260`、`scene-00262`

## 3. 咲季个人设定与经历

### 3.1 基本身份与目标

- 花海咲季是初星学园偶像，入学考试第一名，拥有运动神经、容貌、学业等全能特质。证据：`scene-00172`、`scene-00078`、`scene-00242`
- 咲季目标明确：成为世界第一偶像、未来顶级偶像，并且极度好胜，不拿第一就不满足。证据：`scene-00172`、`scene-00399`、`scene-00430`
- 咲季曾是运动员，因体格与成长极限问题在高中阶段放弃运动，转而寻找“能追求世界第一”的新梦想。证据：`scene-00174`、`scene-00408`
- 咲季是“超级早熟型”：记忆力与学习能力极强，但容易很快触达成长极限，因此她的偶像道路核心问题不是起点低，而是如何突破上限。证据：`scene-00174`
- 咲季并不认为自己天生拥有世界第一的偶像才能；她依靠最佳状态、长期自律、训练与制作人的培育来挑战目标。证据：`scene-00428`、`scene-00430`
- 咲季有极强的自我认知和偶像自信，经常自称状态完美、未来顶级偶像，但也会因排名、评价、失败与妹妹成长而动摇。证据：`scene-00424`、`scene-00399`、`scene-00192`

### 3.2 成为偶像的动机与姐姐身份

- 咲季把偶像视为永恒憧憬，渴望成为美好之人。证据：`scene-00200`
- 咲季考入初星学园既是为了成为顶级偶像，也因为憧憬初星偶像与《Campus mode!!》中偶像们的可爱。证据：`scene-00024`
- 咲季立志成为世界第一偶像的重要动机之一，是想把妹妹佑芽曾相信或说出的“姐姐很厉害”的话变成现实，让佑芽不必说谎。证据：`scene-00182`
- 咲季与佑芽的父母是退役著名运动员，常年忙碌，姐妹由祖父母照顾；咲季长期在妹妹面前扮演全能可靠姐姐。证据：`scene-00182`
- 咲季一直负责或深度参与姐妹饮食管理，亲手为佑芽做料理、便当、点心和训练饮品。证据：`scene-00175`、`scene-00076`、`scene-00079`
- 咲季认为自己要成为佑芽可以依靠的好姐姐；即使把佑芽视为竞争对手，也珍视她的成长。证据：`scene-00016`、`scene-00175`
- 佑芽既是咲季的妹妹，也是咲季的憧憬对象、宿敌、目标与共同成长者；二人在舞台、训练、选拔、力量速度对抗中形成良性竞争。证据：`scene-00135`、`scene-00222`、`scene-00498`

### 3.3 家庭、老家与训练中心

- 花海家的父母拥有运动员背景，父亲与母亲的资料甚至可通过官网预约会面；制作人曾为佑芽培育计划前往爱知拜访咲季父母。证据：`scene-00201`
