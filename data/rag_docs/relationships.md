# 咲季关系知识库


## 入库注意事项

- `scene_id` 是证据索引，可回查 `outputs/scene_summaries.jsonl` 和 `data/rag_docs/scenes/`。
- `H.I.F`、`N.I.A`、`FINALE`、`一等星`、`Re;IRIS`、`Begrazia` 等术语保留原文；官方译名如需发布请二次核对。
- `求婚`、`禁忌之恋`、`疑似交往` 等内容仅作为特定剧情张力或误会，默认不要固化为长期恋爱设定。
- `姐姐/妹妹` 称呼仅在花海咲季与花海佑芽之间作为亲属关系处理；其他角色相关表达需按语境视为照顾、玩笑或称呼梗。
- AU/活动限定设定可用于性格投射，不应覆盖主线世界观。

## 编写原则

- 本文只收录以花海咲季为中心、可反复检索的稳定关系设定、剧情事实与关系定位。
- 重要结论后保留 `scene_id` 作为证据；同一结论优先使用多场景或关键剧情佐证。
- 单场景的玩笑、误会、梦境或活动限定设定，不作为长期关系基线。例如“禁忌之恋”“求婚”“疑似交往”等表述只作为特定剧情中的情感张力或误会记录，不直接固化为长期恋爱设定。（证据：`scene-00015`、`scene-00016`、`scene-00017`、`scene-00044`）
- “姐姐”称呼只在咲季与花海佑芽之间作为亲属关系处理；其他角色口中的“姐姐”“妹妹”等若缺乏稳定亲属证据，按照顾、憧憬、玩笑或角色定位处理。

## 咲季 与 制作人

### 稳定定位

- 咲季与制作人的长期稳定关系是“偶像与制作人”“担当偶像与培育者”“共同追梦的搭档”。制作人负责发掘、指导、歌曲与企划安排，咲季则将其视为能理解自己、支持自己继续挑战的人。（证据：`scene-00012`、`scene-00027`、`scene-00032`、`scene-00172`、`scene-00193`）
- 制作人选中咲季的核心理由，不是她一开始就无敌，而是她即使梦想被粉碎也会重新站起、继续追逐目标的不屈性。（证据：`scene-00027`、`scene-00174`、`scene-00192`）
- 咲季多次确认制作人是自己的“最棒搭档”，并认为两人共同培育了“偶像·花海咲季”。（证据：`scene-00032`、`scene-00192`、`scene-00193`、`scene-00195`）

### 搭档与工作关系

- 制作人为咲季准备原创歌曲、演出企划、训练方案与新曲改写，咲季会主动要求制作人辅导、协助、见证自己的成长。（证据：`scene-00012`、`scene-00013`、`scene-00036`、`scene-00037`、`scene-00491`）
- 在咲季面对佑芽、手毬、琴音、星南等强敌时，制作人的角色是战术制定者、心理支撑者和成长观察者。（证据：`scene-00034`、`scene-00177`、`scene-00180`、`scene-00193`、`scene-00491`）
- 制作人不仅培养咲季的技术，也反复引导她面对“怕输”“逞强”“无法接受失败”等内在问题。（证据：`scene-00014`、`scene-00179`、`scene-00192`、`scene-00195`、`scene-00199`）

### 情感张力与信任

- 两人关系带有强烈的亲密和情感投入：咲季会向制作人暴露脆弱，制作人也会承认对担当偶像投入过多感情。（证据：`scene-00029`、`scene-00177`、`scene-00192`、`scene-00199`）
- “求婚”“禁忌之恋”“约会”等语句和误会属于特定剧情中的情感张力、玩笑或事件表现，不应单独写成长期恋爱设定。（证据：`scene-00015`、`scene-00016`、`scene-00017`、`scene-00021`、`scene-00044`）
- 咲季会表现出对制作人的独占欲与依赖，但稳定设定仍应优先归纳为“深厚信任的工作搭档/担当关系”。（证据：`scene-00030`、`scene-00032`、`scene-00193`、`scene-00195`）

### 竞争与游戏化互动

- 咲季有时会把制作人拉入“对决”或“胜负”式互动，例如暑假休养方案、游乐设施或训练方案争执；这些更像她好胜性格的延伸，不宜写成长期竞争关系。（证据：`scene-00018`、`scene-00019`、`scene-00020`）

### 外部评价

- 制作人眼中的咲季是无论放入哪个组合都能点燃他人的偶像，也是能与手毬、琴音构成强力组合的核心。（证据：`scene-00049`、`scene-00056`）
- 在佑芽线中，制作人也将咲季视为必须保护、无法放弃的偶像，并以她的演出作为重新振作与特训的动力之一。（证据：`scene-00223`、`scene-00227`）

## 咲季 与 花海佑芽

### 亲情

- 咲季是花海佑芽的姐姐，佑芽是咲季的妹妹；这是两人关系的基础设定。（证据：`scene-00173`、`scene-00201`、`scene-00204`、`scene-00208`、`scene-00475`）
- 咲季长期以“可靠姐姐”的身份照顾佑芽，包括饮食、学习、训练、按摩和情绪支持；她也把“成为佑芽可以依靠的姐姐”视为重要目标。（证据：`scene-00175`、`scene-00176`、`scene-00177`、`scene-00206`）
- 咲季的“姐姐”意识与家庭背景相关：父母常年忙碌，咲季在妹妹面前承担全能可靠的角色，并希望把佑芽眼中的“姐姐很厉害”变成现实。（证据：`scene-00182`）

### 竞争

- 佑芽是咲季最重要、最持久的竞争对手；咲季也长期是佑芽想战胜的目标。姐妹间的竞争贯穿选拔考试、H.I.F、N.I.A、FINALE 等关键舞台。（证据：`scene-00173`、`scene-00179`、`scene-00183`、`scene-00208`、`scene-00211`、`scene-00222`）
- 咲季一方面害怕输给妹妹，一方面又期待佑芽成长到能威胁自己；这种矛盾是两人关系的核心张力。（证据：`scene-00176`、`scene-00179`、`scene-00180`、`scene-00208`）
- 佑芽最终追上并战胜咲季后，两人的关系从单向追赶变成互为目标。咲季承认败北、不甘心，也决定不再独自逞强，继续与佑芽再战。（证据：`scene-00194`、`scene-00195`、`scene-00220`、`scene-00222`）
- 在 Re;IRIS 与 Begrazia 对抗阶段，姐妹竞争扩展为组合对组合的竞争；咲季把与佑芽的胜负放入团队战框架，最终与手毬、琴音一起取得冠军。（证据：`scene-00481`、`scene-00482`、`scene-00491`、`scene-00496`、`scene-00497`、`scene-00499`）

### 搭档与相互成就

- 咲季不仅是佑芽的对手，也是佑芽爆发性成长的关键契机；制作人明确评价咲季既是佑芽的最强宿敌，也是可靠的姐姐。（证据：`scene-00204`）
- 咲季会协助制作人准备“向咲季挑战”的佑芽新曲，并把这视为让佑芽成长的方式；佑芽也将此理解为与姐姐并肩作战。（证据：`scene-00043`、`scene-00045`）
- 当佑芽败给燐羽时，咲季会以姐姐身份出面安慰并为妹妹复仇。（证据：`scene-00212`、`scene-00213`、`scene-00187`）

### 团队互动

- 咲季和佑芽常出现在同一社交圈中，围绕料理、学习、礼物、训练、选拔考试等事件展开互动。（证据：`scene-00093`、`scene-00131`、`scene-00155`、`scene-00206`、`scene-00475`、`scene-00476`）
- 在后期组合线中，佑芽代表 Begrazia 向咲季所在的 Re;IRIS 提出对战申请，姐妹关系与组合竞争并行。（证据：`scene-00481`、`scene-00482`、`scene-00486`、`scene-00497`）

### 外部评价

- 制作人、星南等外部角色多次将咲季定位为佑芽的最大目标、宿敌和成长标尺。（证据：`scene-00204`、`scene-00205`、`scene-00210`、`scene-00254`、`scene-00474`、`scene-00486`）
- 佑芽本人认为世界第一的偶像就是姐姐，这说明咲季在佑芽心中既是亲人，也是偶像理想的具象化。（证据：`scene-00205`、`scene-00210`）

### 不应固化的误会

- 佑芽误会咲季与制作人交往、吃醋或把亲密互动理解成恋爱，是具体剧情中的误会，不应写作稳定三角恋设定。（证据：`scene-00015`、`scene-00044`、`scene-00045`）

## 咲季 与 藤田琴音

### 稳定定位

- 咲季与琴音是朋友、同僚、竞争对手，并在 Re;IRIS 中成为队友。（证据：`scene-00031`、`scene-00033`、`scene-00058`、`scene-00497`、`scene-00499`）
- 琴音常把咲季视为强大的顶尖偶像或需要追上的对象；咲季也认可琴音的实力，把她放在“强劲对手/重要朋友”的位置上。（证据：`scene-00007`、`scene-00031`、`scene-00033`、`scene-00168`）

### 竞争

- 咲季、琴音、手毬三人常以“互相竞争又互相利用成长”的关系出现；合练、共演和正式演出都可以同时带有协作和竞争性质。（证据：`scene-00031`、`scene-00033`、`scene-00034`、`scene-00496`）
- 琴音知道自己在技术或经验上与咲季、手毬存在差距，因此常把对她们的仰望、压力和不甘转化为成长动机。（证据：`scene-00007`、`scene-00008`、`scene-00168`）

### 搭档与团队互动

- 咲季、琴音、手毬组成 Re;IRIS 的核心三人关系；三人既是队友，也是正式演出当天仍要互相较量的竞争对手。（证据：`scene-00481`、`scene-00482`、`scene-00496`、`scene-00497`、`scene-00499`）
- 琴音在咲季与手毬发生冲突时可承担调解者角色，是三人关系中重要的缓冲与组织者。（证据：`scene-00058`、`scene-00481`）
- 后期 Re;IRIS 中，琴音被明确写作队长；咲季在队伍里承担强势推进、点燃胜负心和提供料理/训练支持的角色。（证据：`scene-00481`、`scene-00482`、`scene-00497`）

### 外部评价

- 制作人认为咲季、手毬、琴音能构成“最强组合”或高潜力组合，尤其咲季能激发其他偶像燃烧起来。（证据：`scene-00049`、`scene-00056`）
- 手毬评价咲季是真正自律、能持续努力并乐于分享诀窍的人；这也解释了咲季能在三人关系中影响琴音和手毬的原因。（证据：`scene-00058`）
- 星南与其他角色也将琴音和咲季放在高水平竞争者语境中讨论，说明琴音对咲季不是单纯后辈或粉丝，而是会进入同一胜负场的对手。（证据：`scene-00168`、`scene-00232`）

### 不应固化的内容

- 琴音梦见咲季照顾自己、手毬教自己唱歌，是梦境内容，可用作琴音心理与愿望的旁证，但不能写成现实中长期发生的照顾关系。（证据：`scene-00006`）
- “咲季是琴音的姐姐”等表述若缺乏亲属证据，应视为照顾者、前辈感或玩笑式称呼，不写入亲属关系。

## 咲季 与 月村手毬

### 稳定定位

- 咲季与手毬是朋友、队友、竞争对手；两人关系里同时存在互相认可、互相吐槽和正面冲突。（证据：`scene-00031`、`scene-00033`、`scene-00058`、`scene-00059`、`scene-00497`）
- 手毬认可咲季是真正自律、持续努力、愿意分享诀窍的“禁欲系偶像”，这构成她对咲季的核心评价。（证据：`scene-00058`）

### 竞争

- 咲季会把手毬和琴音视为演唱会上的对手，也会在共同训练时把彼此实力当作可利用、可对抗的成长资源。（证据：`scene-00031`、`scene-00033`、`scene-00034`、`scene-00496`）
- 手毬认为咲季和琴音同样不服输，三人性格相似，因此能形成既容易碰撞又容易合拍的关系。（证据：`scene-00059`）

### 搭档与团队互动

- 咲季、手毬、琴音共同构成 Re;IRIS；队内既有竞争，也有明确的队友身份和共同夺冠目标。（证据：`scene-00481`、`scene-00482`、`scene-00495`、`scene-00497`、`scene-00499`）
- 手毬曾因炸猪排面衣等小事闹情绪，咲季会直接制止、安抚或补救；这种互动体现亲近队友间的日常摩擦，不应写成严重敌对。（证据：`scene-00476`、`scene-00481`）
- 咲季也会因手毬演唱会前夜任性打电话而严厉斥责，体现她维护团队秩序和照顾同伴健康的责任感。（证据：`scene-00169`）

### 外部评价

- 制作人认为手毬与咲季、琴音的组合有让手毬正常发挥的可能，是值得关注的组合形态。（证据：`scene-00056`）
- 手毬明确说咲季和琴音是特别的，不是队伍负资产；这说明在 Re;IRIS 中，手毬对咲季的认可已超出普通队友评价。（证据：`scene-00495`）

### 不应固化的内容

- 手毬梦中的咲季、琴音组合是梦境材料，不直接作为现实关系设定。（证据：`scene-00056`）
- “咲季属于手毬”等表达属于手毬当场情绪或夸张语境，可作为亲密与占有欲旁证，不写成正式关系设定。（证据：`scene-00059`）

## 咲季 与 十王星南

### 稳定定位

- 星南是咲季的前辈、学生会长与“一等星”级目标；咲季对星南有尊敬、期待和挑战欲。（证据：`scene-00155`、`scene-00160`、`scene-00162`、`scene-00253`）
- 咲季与星南是镜像型竞争关系：制作人指出咲季像星南不顾一切、青涩炽热的另一个自己；星南也认为自己能理解咲季当前的心情。（证据：`scene-00049`、`scene-00494`）

### 竞争

- 咲季会把星南视为需要超越的高墙，反驳星南关于偶像成长极限的悲观论，并宣言要粉碎那道墙。（证据：`scene-00253`）
- 在 H.I.F、初星偶像祭等关键舞台上，星南和咲季处于争夺“一等星”或顶级偶像称号的竞争关系。（证据：`scene-00231`、`scene-00494`、`scene-00497`）
- 星南并非单纯反派或上位压迫者。她能理解咲季的混乱与压力，也承认咲季是全世界最能与自己心情相照的人之一。（证据：`scene-00494`）

### 搭档与同盟

- 咲季可与星南、燐羽组成同盟，围绕新曲、特训、项目『星屑』和挑战一等星展开合作。（证据：`scene-00038`、`scene-00196`、`scene-00197`、`scene-00219`）
- 咲季在考察星南是否适合培育佑芽时，一开始保持审慎，但最终认可星南作为学生会长和榜样的出色之处。（证据：`scene-00160`、`scene-00162`）

### 团队互动

- 星南常与佑芽、美铃、Begrazia、学生会等团队线相连；咲季则在 Re;IRIS 与 Begrazia 的对战中与星南再次形成团队对抗结构。（证据：`scene-00474`、`scene-00483`、`scene-00486`、`scene-00491`、`scene-00494`）
- 星南、燕、莉波等前辈的评价会反衬咲季的成长：燕和莉波指出咲季的说话方式和爱说大话像过去的星南。（证据：`scene-00253`）

### 不应固化的内容

- 机器人、基地、Mother 等特殊设定场景属于活动/架空剧情时，不应覆盖正篇中的学园前辈、会长、劲敌关系。（证据：`scene-00011`、`scene-00054`）

## 其他人物关系补充

### 咲季 与 麻央

- 麻央主要作为宿舍前辈、宿舍长或三年级生，与咲季共享初星寮团队日常。咲季常被放在“琴音、咲季、手毬”的热闹三人组或宿舍训练成员中。（证据：`scene-00003`、`scene-00159`、`scene-00247`）
- 咲季会参与宿舍活动提案、训练、慰问等集体行动；与麻央的关系更多是同宿舍成员/后辈与前辈的团队互动，而非核心个人线。（证据：`scene-00001`、`scene-00002`、`scene-00288`）
